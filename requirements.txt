# Python dependencies for this project
#
# This project currently uses only the Python standard library.
# Install with:
#   pip install -r requirements.txt
#
# Required Python runtime:
#   Python 3.12+
#
# Required system tools for video features:
#   1. ffmpeg  - used by sender.py for MPEG-TS preprocessing and by receiver.py for remux
#   2. ffplay  - used by receiver.py for real-time video playback
#
# Make sure ffmpeg.exe and ffplay.exe are both available in PATH.
